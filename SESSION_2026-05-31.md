## If Certificate Warning Appears

If the browser shows a certificate warning for `www.jjhsmartweddings.com`, wait 30-60 minutes first.

If it still does not clear:

1. Open Vercel project settings:

```text
Vercel -> wedding -> Settings -> Domains
```

2. Check that both domains show valid configuration:

```text
jjhsmartweddings.com
www.jjhsmartweddings.com
```

3. Click `Refresh` or `Verify` in Vercel if available.

4. Confirm Cloudflare DNS is still DNS only:

```text
CNAME  www  cname.vercel-dns.com  DNS only
```

5. Make sure the record is not proxied through Cloudflare.

6. If still stuck, remove `www.jjhsmartweddings.com` from Vercel Domains and add it again to trigger SSL provisioning again.

7. If Cloudflare has CAA records in the future, make sure they allow Let's Encrypt or Vercel certificate issuance.
