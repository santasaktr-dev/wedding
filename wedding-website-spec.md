# Wedding Website Spec

Responsive one-page wedding website for Jah & Smart.

Source details are normalized from `wedding_website_implementation_plan_md.md`.

## Core Details

- Couple: Jah & Smart
- Date: Sunday, 1 November 2026
- Time: From 16:00 onwards
- Venue: Pearl Wedding Venue
- Theme: Modern Old Money / Minimal Luxury
- Dress code: Old Money Elegance

## MVP Sections

1. Hero
2. Event Info
3. Schedule
4. Location
5. Dress Code
6. RSVP
7. Contact
8. Footer

## Design Direction

- Oxford Navy: `#0A1F44`
- Tweed Brown: `#7C5C3B`
- Deep Olive: `#3E4D3A`
- Camel Beige: `#D6C8A5`
- Ash Grey: `#BDBFBA`

The visual tone should be refined, minimal, premium, and uncluttered.

## RSVP

For MVP, RSVP is a polished front-end form only. Backend integration is intentionally deferred.

Future integration options:

- Google Form
- Google Sheets integration
- Supabase or Firebase if a full guest system is needed later
