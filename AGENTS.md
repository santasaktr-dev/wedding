# Project Instructions

This project is a responsive wedding website for Jah & Smart.

Use `wedding-website-spec.md` as the main product specification.

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Mobile-first responsive design

## Design Direction

- Modern Old Money
- Oxford Navy, Tweed Brown, Deep Olive, Camel Beige, Ash Grey
- Minimal, elegant, premium, not cluttered

## Requirements

- Website must work well on mobile and desktop
- Use semantic HTML
- Keep components clean and reusable
- Avoid overly complex backend unless requested
- Prioritize MVP sections first:
  1. Hero
  2. Event Info
  3. Schedule
  4. Location
  5. Dress Code
  6. RSVP
  7. Contact
  8. Footer

## Done Means

- Responsive layout tested on mobile and desktop
- No obvious layout breaking
- All main sections implemented
- Code is clean and easy to edit
